Crystal Game

  1. You will be given a random number at the start of the game.
  2. There are four crystals below.
    • Blue
    • Green
    • Red
    • Yellow
  3. By Clicking on a crystal, you will add a specific amount of points to your total score.
  4. You win the game by matching
  5. You lose if you don't match
  6. The value adds up
  7. Each time the game starts, new values will start
